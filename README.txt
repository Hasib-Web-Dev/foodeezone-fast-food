FoodeeZone — Setup
====================

Files:
- index.html  → main page (Tailwind via CDN)
- styles.css  → custom CSS (thin scrollbar)
- script.js   → all interactivity (cart, filters, reviews, etc.)

How to run:
1) Put all files in the same folder.
2) (Optional) Add your own logo images named:
   - "FoodeeZone PNG.png" (header circle logo)
   - "FoodeeZone.jpeg" (footer small logo)
   If you don't have these, edit the <img> src paths in index.html.
3) Open index.html in your browser.

Notes:
- LocalStorage keys used: "budget-foodget.cart" and "budget-foodget.reviews".
- Delivery is free over ৳299; otherwise ৳40. Modify in script.js if needed.
